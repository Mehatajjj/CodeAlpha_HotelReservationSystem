# Hotel Reservation System

## Description
This is a Java console-based Hotel Reservation System developed for the CodeAlpha Internship.

The system allows users to:
- View available rooms
- Book rooms
- Cancel reservations
- Simulate payment process

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner

## Room Categories
- Standard
- Deluxe
- Suite

## How to Run

1. Compile all files:
   javac Room.java Booking.java HotelReservationSystem.java

2. Run:
   java HotelReservationSystem

- Classes & Objects
- Encapsulation
- Collections (ArrayList)
- Loops & Switch Case
